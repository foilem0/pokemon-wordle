# Pokémon Wordle

Python app for a Pokémon themed Wordle

Based on:
- New York Time's Wordle
- The RESTful PokéAPI
## Features

- PokéAPI integration
- 2 separate game modes
- A hint system
- SQLite3 module for SQLite local database usage
- Cached selection pool for better performance and ethical API usage
